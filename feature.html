1サイトの解説
サイト名：
Airbnb Clone


製作期間：1ヶ月

使用したもの：
HTML / CSS / Bootstrap / Javascript / jQuery / Ruby / Ruby on Rails / mySQL / Git / Github / Docker / AWS / /Lucidchart / Heroku

コメント：

今回のプロジェクトはAirbnbのような旅行者と宿泊施設をマッチングするアプリケーションを開発しました。今回は実際の開発現場を想定して開発環境でDockerとGithubを使い、より現場に近い形で開発を行ってきました。

以前に開発したElearningよりさらに発展したものを作成するため、今回はAWSを使用したりRails単体ではなく、Rails × Javascript を掛け合わせて開発しました。

今回難しかった点は、色々な機能を実装したのでRails単体の難しさというよりはRails + α という難しさがありました。

このアプリケーションではfacebook authentication、カレンダーの他にもGoogle Mapなどの機能も実装しておりUIの部分でも工夫してます。
